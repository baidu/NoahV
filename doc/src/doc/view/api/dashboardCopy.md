## 接口含义
复制仪表盘接口

*复制仪表盘需要生成一个与目标仪表盘一致的新仪表盘(主要是仪表盘具体配置一致)，但需要保证id和仪表盘名称不一致*

## 请求参数

| 属性  | 说明         | 类型   | 默认值 |
| ----- | ----------- | ------ | ------ |
| id |    待复制仪表盘接口id | number | ''    |


## 返回数据参数

返回数据整体格式参考[接口规范](#/guide/specification)中的请求成功接口。


res.data

空
