## 接口含义
获取仪表盘组件和布局配置信息

## 请求参数

| 属性  | 说明         | 类型   | 默认值 |
| ----- | ----------- | ------ | ------ |
| id |    仪表盘id | number | ''    |


## 返回数据参数

返回数据整体格式参考[接口规范](#/guide/specification)中的请求成功接口。


res.data
| 属性  | 说明         | 类型   | 默认值 |
| ----- | ----------- | ------ | ------ |
| configure |  仪表盘组件和布局配置新，字符串化的JSON数据 | string | ''    |
| id |  仪表盘id | number | ''    |
